# Projet jeu SIGA - Shajudan Invaders

Projet de jeu pour la matière *Swarm Intelligence and Genetic Algorithm* basé la blibliothèque moteur de jeu [PixiJS](https://pixijs.com/).

## Pour lancer le projet

### Prérequis

- nodejs

### Installation

```bash
npm install
```

### Lancer le projet

```bash
npm run start
```
