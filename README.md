# Projet jeu - SIGA

Projet de jeu pour la matière *Swarm Intelligence and Genetic Algorithm* basé sur le repo [Canva](https://github.com/arthurescriou/canvas) d'Artur Escriou.

## Pour lancer le projet

### Prérequis

- nodejs

### Installation

```bash
npm install
```

### Lancer le projet

```bash
npm run start
```
